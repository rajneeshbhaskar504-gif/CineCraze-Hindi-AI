# 🎬 CineCraze Hindi AI

An AI-powered Movie Explained Script Generator built with Streamlit.

## Features
- 🤖 AI Hindi Script Generator
- 🎬 Movie Search
- 📝 YouTube Title Generator
- 📄 Description Generator
- 🏷️ SEO Tags
- 🌙 Dark Theme
- 🚀 Streamlit Cloud Ready

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
